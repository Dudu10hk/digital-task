**למה זה חובה?**  
בלי זה המערכת לא תוכל לשמור משתמשים חדשים.

### איך עושים את זה?

#### שלב 1: פתח Supabase
1. היכנס ל-https://supabase.com
2. בחר את הפרויקט שלך
3. לחץ על **SQL Editor** בתפריט הצד

#### שלב 2: הרץ את הסקריפטים

**אם זו התקנה ראשונה:**
```
1. פתח את הקובץ: scripts/001_create_tables.sql
2. העתק את כל התוכן
3. הדבק ב-SQL Editor
4. לחץ RUN
5. אחרי שזה מסתיים, עבור לשלב 3
```

**אם הטבלה כבר קיימת:**
```
1. פתח את הקובץ: scripts/007_add_password_column.sql
2. העתק את כל התוכן
3. הדבק ב-SQL Editor
4. לחץ RUN
5. אחרי שזה מסתיים, עבור לשלב 3
```

#### שלב 3: הגדר הרשאות (חובה!)
```
1. פתח את הקובץ: scripts/008_users_rls_policies.sql
2. העתק את כל התוכן
3. הדבק ב-SQL Editor
4. לחץ RUN
5. ודא שאין שגיאות
```

   - אימייל (חובה)

3. **לחץ "הוסף משתמש"**

4. **העתק את הפרטים** שמוצגים:

## ❓ שאלות נפוצות

### "המשתמש נעלם אחרי שהוספתי אותו"
**פתרון:** רענן את הדף. אם זה לא עוזר - ודא שהרצת את סקריפטי ה-SQL.

**פתרון:** בדוק Console (F12) לשגיאות. וודא ש-Supabase מחובר.

**פתרון:** ודא שהעברת את הסיסמה הנכונה בלי רווחים.

1. בממשק: "ניהול משתמשים"
2. ב-Supabase: Table Editor → users
