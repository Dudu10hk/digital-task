# מדריך מהיר - הוספת משתמשים חדשים

## לפני שמתחילים

**חובה להריץ סקריפט אחד ב-Supabase:**

1. היכנס ל-Supabase Dashboard שלך
2. לחץ על SQL Editor (בתפריט הצד)
3. הרץ את אחד הסקריפטים:
   - `scripts/001_create_tables.sql` (אם זו התקנה ראשונה)
   - `scripts/007_add_password_column.sql` (אם הטבלה כבר קיימת)

## איך מוסיפים משתמש?

### שלב 1: פתח את ניהול המשתמשים
- לחץ על הכפתור "ניהול משתמשים" בתפריט העליון

### שלב 2: מלא את הפרטים
- **שם מלא** ✅ (חובה)
- **אימייל** ✅ (חובה)
- **סיסמה** (אופציונלי - אם לא תמלא, הסיסמה תהיה: `[שם_ראשון]123`)
- **תמונת פרופיל** (אופציונלי)
- **הרשאות**: בחר admin / user / viewer

### שלב 3: הוסף משתמש
- לחץ על "הוסף משתמש"
- **שים לב!** תופיע הודעה עם פרטי הכניסה:
  ```
  אימייל: user@example.com
  סיסמה: yossi123
  ```
- **חשוב מאוד:** שמור פרטים אלו! הם לא יוצגו שוב!

### שלב 4: מסור למשתמש החדש
- העבר למשתמש החדש את:
  - האימייל שהוזן
  - הסיסמה (שיצרת או שנוצרה אוטומטית)

## דוגמה

**אם יצרת משתמש בשם "יוסי לוי":**
- אימייל: `yossi@company.com`
- סיסמה (אוטומטית): `יוסי123`

**אם הזנת סיסמה משלך:**
- אימייל: `yossi@company.com`
- סיסמה: `MyPassword123!`

## הערות חשובות

⚠️ **פרטי הכניסה מוצגים רק פעם אחת** - אחרי סגירת ההודעה, לא תוכל לראות אותם שוב

✅ **המשתמשים נשמרים ב-Supabase** - לא רק במצב דמו

🔒 **רק מנהלי מערכת** יכולים להוסיף משתמשים חדשים

## בעיות?

### "שגיאה ביצירת משתמש"
- ודא שהרצת את הסקריפט SQL ב-Supabase
- בדוק שיש לך חיבור לאינטרנט
- נסה לרענן את הדף

### "אימייל כבר קיים"
- האימייל הזה כבר רשום במערכת
- השתמש באימייל אחר או מחק את המשתמש הקיים

### הסיסמה לא עובדת
- ודא שהעברת את הסיסמה הנכונה למשתמש
- בדוק שאין רווחים בהתחלה או בסוף
- נסה ליצור משתמש חדש אם הסיסמה אבדה
