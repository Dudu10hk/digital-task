## 🔧 סקריפטים

### 📄 [setup.sh](setup.sh)
**סקריפט התקנה אוטומטי**

הרצה:
```bash
./setup.sh
# או
npm run setup
```

מה הוא עושה:
- ✅ בודק אם .env.local קיים
- ✅ יוצר אותו אם לא
- ✅ מתקין תלויות
- ✅ מדפיס הוראות

---

### 📄 [init-git.sh](init-git.sh)
**סקריפט אתחול Git**

הרצה:
```bash
./init-git.sh
# או
npm run init-git
```

מה הוא עושה:
- ✅ מאתחל Git repository
- ✅ יוצר .gitignore
- ✅ עושה initial commit
- ✅ מדפיס הוראות לGitHub
