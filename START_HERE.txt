2️⃣  הגדר Supabase:
    - היכנס ל-https://supabase.com
    - צור פרויקט חדש (חינמי)
    - הרץ את scripts/002_create_task_tables.sql

3️⃣  הגדר משתני סביבה:
    - ערוך את .env.local
    - הוסף את ה-URL וה-Key מSupabase

4️⃣  צור משתמש Admin:
    - Supabase > Authentication > Add user
    - User Metadata: {"name": "שם", "role": "admin"}

5️⃣  הרץ!
    npm run dev
    
    פתח: http://localhost:3000
